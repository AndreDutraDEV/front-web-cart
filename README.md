# 🍰 Product List with Cart

Este é um projeto de uma interface de listagem de produtos com funcionalidade de carrinho de compras. Ele foi desenvolvido como parte de um desafio do [Frontend Mentor](https://www.frontendmentor.io?ref=challenge) para praticar habilidades em HTML, CSS e JavaScript.

## 📸 Visão Geral

O projeto apresenta uma vitrine de sobremesas com imagens, preços e a opção de adicionar produtos ao carrinho. O usuário pode:

- Visualizar produtos com imagens e descrições.
- Adicionar e remover produtos do carrinho.
- Alterar a quantidade de itens diretamente no carrinho ou na listagem.
- Ver o valor total da compra.
